Trend Report

Lighthouse Labs Final Project

March 2016

Trend Report was built using Ruby, Ruby on Rails, PostgreSQL, HTML5, CSS3 and jQuery, with partner Sandeep Samra.

It is a web application that scrapes trendy blogs and websites and provides articles for different categories including food and fashion. The purpose of this application is to allow a user to access articles related to a certain category without having to go to each individual blog or website. Stuff I learned: how to work with a bootsrap based theme, the importance of mobile optimizations, and you definitely shouldn't use a hard coded web-scraper 'cause it won't ever run again after deployment (background workers would have been a better choice).
