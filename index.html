Ruby On Rails App

This project was built using Ruby, Ruby on Rails, PostgreSQL, HTML5, CSS3 and jQuery, with Lighthouse Labs classmate as our final project. Trend Report is a web application that scrapes trendy blogs and websites and provides articles from different categories including food and fashion. What I learned: how to work with a bootstrap based theme, the importance of mobile optimizations, and you definitely shouldn't use a hard coded web-scraper 'cause it won't ever scrape again after deployment (background workers would have been a better choice).
